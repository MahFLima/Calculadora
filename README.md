# ⚙️ Instalação

```bash
# Clone o repositório
$ git clone https://github.com/Mahflima/calculadora.git

# Entre no repositório
$ cd calculadora

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev
```
